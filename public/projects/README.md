# 作品图片放置说明

请将你的真实作品图片放到本目录，并使用以下固定文件名：

1. `project-1-history-game.png`
2. `project-2-internship-affiliate.png`
3. `project-3-history-video.png`
4. `project-4-timebook-prototype.png`

建议尺寸：
- 横图比例：16:10 或 16:9
- 建议宽度：1600px 以上
- 文件格式：PNG / JPG / WebP（若不是 `.png`，请同步修改 `data/projects.ts` 中 `cover` 路径）

简历文件请放到：
- `/public/resume.pdf`
